Class Bond::Input

  1. lib/bond/input.rb
Parent: String

A string representing the last word the user has typed. This string is passed to a mission action to generate possible completions. This string contains a number of attributes from the matching mission, useful in generating completions.

Attributes

argument [RW] Current argument number and array of argument strings. Used by MethodMission.
arguments [RW] Current argument number and array of argument strings. Used by MethodMission.
line [R] The full line the user has typed.
matched [R] MatchData object from the matching mission (Mission#matched).
object [RW] Actual object a user has just typed. Used by MethodMission and ObjectMission.
