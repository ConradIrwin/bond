anywhere_mission.rb

lib/bond/missions/anywhere_mission.rb
Last Update: Tue Apr 27 05:17:10 -0400 2010

A mission which completes anywhere i.e. even after non word break characters such as ’[’ or ’}’. It generates the following regexp condition /#{prefix}(#{anywhere})$/ and passes the first capture group to the mission action.

Bond.complete Options:

:anywhere
A regexp string which generates the first capture group in the above regexp.
:prefix
An optional string which prefixes the first capture group in the above regexp.
